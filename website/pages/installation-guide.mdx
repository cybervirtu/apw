## What this covers

<Steps>
### Check the basic tools

Make sure you have Git, Bash or a compatible shell, and local terminal access.

### Clone APW locally

Clone the APW framework repo into your workspace and enter the APW root.

### Verify APW works

Run `git --version`, `bash --version`, and `./apw help`.

### Optional docs portal setup

Only if you want the rendered visual docs experience, install the `website/` dependencies and run the portal locally.
</Steps>
