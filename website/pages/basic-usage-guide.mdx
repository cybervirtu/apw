## What this covers

<Steps>
### Create your first project

Use Create Project to make the downstream repo safely.

### Initialize project state

Run the guided initializer so the core `.gsd` files tell a real project story.

### Open the downstream project

Move into the real project repo and open that folder in your IDE.

### Read `AGENTS.md` and run First Run

Use the in-IDE front door, then use the short first-run checklist.

### Choose the first workflow

Use `/brainstorm`, `/create`, `/orchestrate`, or `/status` based on the real situation.
</Steps>
