## APW contract files

These files define the APW standard. They are copied into new projects to establish the governing baseline.

| File | Purpose |
| --- | --- |
| `PROJECT_RULES.md` | mandatory execution rules |
| `AGENT_SYSTEM.md` | dual-engine and precedence model |
| `COMMAND_POLICY.md` | command ownership rules |
| `PROJECT_BOOTSTRAP.md` | bootstrap and upgrade contract |
| `AGENTS.md` | tool-facing front door |
