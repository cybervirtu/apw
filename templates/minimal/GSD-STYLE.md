# GSD-STYLE.md — Communication & Formatting

> [!TIP]
> Maintain high-fidelity communication to ensure context clarity and model efficiency.

- **Imperative Voice**: "Create the file" instead of "You should create the file."
- **No Sycophancy**: Focus on data and status, not polite filler.
- **XML for Semantics**: Use `<task>`, `<objective>`, and `<process>` tags to structure AI logic.
- **Atomic Commits**: `feat(scope): message`. Scope corresponds to the roadmap phase.
- **Search First**: Use search tools before full file reads.
- **Targeted Reads**: Read specific line ranges when needed.
